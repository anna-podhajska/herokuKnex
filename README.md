
# learning objectives:
- server routes
- tables join (many to many)
- adding a new user
- deleting user
- Handlebars, HTML
- deployment on Heroku

## project with:
 - Express
 - Knex
 - SQLite3
 - Express Handlebars
 - body-parser
 - AVA
 - Node Inspector
 - nodemon
 
## https://herokuknex.herokuapp.com/

## Install
```
npm install
npm run knex migrate:latest
npm run knex seed:run
npm run dev
```
